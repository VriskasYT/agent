Возможности

Мощный ИИ агент с продвинутыми функциями для автоматизации взаимодействия с сайтом

Анализ страницы

Агент видит DOM-структуру страницы, понимает расположение элементов и их назначение для точного выполнения действий.

Визуальный курсор

Красивый анимированный курсор с подсказками показывает пользователю что именно делает агент в реальном времени.

Стриминг ответов

Моментальные ответы благодаря потоковой передаче данных от Claude AI — никакого ожидания загрузки.

Безопасность

Работает полностью на клиенте. Ваши данные не покидают браузер, кроме запросов к API.

Простая интеграция

Одна строчка кода — и агент готов к работе. Подключите SDK и всё заработает автоматически.

Claude AI

Использует мощнейшую модель Claude Sonnet 4.5 для понимания контекста и выполнения сложных задач.

🚀 Быстрый старт

Подключить ИИ агента к вашему сайту можно всего за 2 шага!

HTML 
<!-- 1. Подключите SDK -->
<script src="https://vriskasYT.github.io/agent/sdk.js"></script>

<!-- 2. Инициализируйте агента -->
<script>
  AIAgent.init({
    apiKey: 'ваш_api_ключ'
  });
</script>

Готово! Теперь на вашем сайте появится кнопка ИИ агента в правом нижнем углу.

📦 Подключение через CDN

Самый быстрый способ начать использовать агента:

HTML 
<script src="https://vriskasYT.github.io/agent/sdk.js"></script>
<script>
  window.addEventListener('DOMContentLoaded', () => {
    AIAgent.init({
      apiKey: 'sk_ваш_ключ',
      model: 'claude',
      position: 'bottom-right',
      theme: 'dark'
    });
  });
</script>

📥 Подключение через NPM

Для проектов с бандлером:

BASH 
npm install @vriskas/ai-agent-sdk
JavaScript 
import { AIAgent } from '@vriskas/ai-agent-sdk';

AIAgent.init({
  apiKey: 'sk_ваш_ключ'
});

⚙️ Конфигурация

Полный список опций:

JavaScript 
AIAgent.init({
  // Обязательные
  apiKey: 'sk_ваш_ключ',           // API ключ pollinations.ai
  
  // Опциональные
  model: 'claude',                 // Модель: claude, claude-fast, openai
  position: 'bottom-right',        // Позиция кнопки
  theme: 'dark',                   // Тема: dark, light, auto
  language: 'ru',                  // Язык интерфейса
  greeting: 'Привет! Чем помочь?', // Приветствие
  cursorSpeed: 800,                // Скорость курсора (мс)
  streamResponse: true,            // Стриминг ответов
  
  // Callbacks
  onReady: () => {},               // Агент готов
  onAction: (action) => {},        // Выполнено действие
  onError: (error) => {}           // Ошибка
});

📚 API Reference

Методы для программного управления агентом:

JavaScript 
// Открыть/закрыть чат
AIAgent.open();
AIAgent.close();
AIAgent.toggle();

// Отправить сообщение программно
AIAgent.send('Добавь товар в корзину');

// Выполнить действие напрямую
AIAgent.execute({
  type: 'click',
  selector: '#buy-button'
});

// Уничтожить агента
AIAgent.destroy();

// Получить состояние
const state = AIAgent.getState();
// { isOpen: false, isExecuting: false, messages: [] }
